# Model Validation

-   **/st_runs_50**: Validation water temperatures at headwater and outlet gages for the top 50 calibration runs of configurations M1, M2, M3, and M4.
-   **M1_calibration.csv**: Sampled parameter values and model outcomes for the top 50 validation runs of M1.
-   **M2_calibration.csv**: Sampled parameter values and model outcomes for the top 50 validation runs of M2.
-   **M3_calibration.csv**: Sampled parameter values and model outcomes for the top 50 validation runs of M3.
-   **M4_calibration.csv**: Sampled parameter values and model outcomes for the top 50 validation runs of M4.
-   **mod_time_pst_val.csv**: Model timestamps used to align with validated water temperature predictions.
